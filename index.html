Análisis de Red Neuronal Artificial

1Análisis de colab, construcción de un modelo de clasificación empleando redes neuronales

El colab "Construcción de un modelo de clasificación empleando redes neuronales" explica una red neuronal artificial. Una neurona artificial es un algoritmo, es decir, un conjunto de pasos precisos para resolver un problema, que genera una respuesta o salida única para una entrada(x).

2Elementos de la neurona artificial

Entradas: Las entradas corresponden a las dendritas de una neurona biológica. Son los datos que recibe una Neurona artificial. Los pesos sinápticos(Weights) corresponden al lugar donde se almacena el intercambio de información entre las neuronas artificiales. En una neurona biológica se establecen sinapsis entre las dendritas de la neurona y el axón. La suma ponderada sobre los elementos de la entrada, corresponden al procesamiento que se obtiene de un potencial postsináptico (valor que es función de las entradas y los pesos sinápticos). Esta suma ponderada también se conoce como regla de propagación. En este sentido, se produce una función de activación, o también llamado “la neurona se activó”, que significa el valor obtenido con la regla de propagación, se filtra a través de una función conocida como función de activación y es la que nos da la salida de la neurona. Esta nos permite lograr y potenciar el objetivo deseado.

3Red neuronal artificial

Una red neuronal artificial siempre está compuesta por una capa de entrada, una capa de salida (si solo hay una capa en la red neuronal, la capa de entrada coincide con la capa de salida) y puede contener 0 o más capas ocultas.

4¿Cómo se entrena una Red neuronal artificial?

Entrenar una red neuronal consiste en ajustar cada uno de los pesos de las entradas de todas las neuronas que forman parte de la red neuronal, para que las respuestas de la capa de salida se ajusten lo más posible a los datos que conocemos.

5Modelos regresivos

Los modelos regresivos son tipos de modelo que predicen una etiqueta (label) de datos numéricos, es decir, datos que contienen números (Cantidad de personas, peso, precios). La salida de un modelo de regresión es un número, como el peso de un animal.

6Regresión lineal

La regresión lineal es un método amplio(widely)usado para estimar valores, como el precio de un apartamento, el valor de un cierto objeto(Stock), el valor de expectativa de vida de un individuo, etc., La regresión lineal se ve aplicada como una cantidad excesiva(plethora-plétora) de fórmulas que incluyen derivadas, sistemas de ecuaciones, y determinantes.

7Conclusiones

En conclusión, una red neuronal es un modelo simplificado que sigue el modo en que el cerebro humano procesa la información: Funciona simultaneando un número elevado de unidades de procesamiento interconectadas que parecen versiones abstractas de neuronas. (IBM, 2021). Machine learning es una herramienta que nos ayuda a analizar datos, formular algoritmos y brindar situaciones a través de una red neuronal artificial que imita el funcionamiento del cerebro humano.
